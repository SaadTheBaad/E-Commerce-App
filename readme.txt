These dependencies were used:
npm install react-router-dom
npm install pip install Flask
pip install flask-cors in the backend directory

Steps (NOTE THESE STEPS WERE MADE FOR WINDOWS): 
1. Run npm install in the terminal
2. After all dependency are installed, open a new terminal and make your way to the backend directory
3. Run this command: .\venv\Scripts\activate 
4. Run this comman next: python app.py
5. Now the backend should be running
6. Next go back to the old terminal where you ran npm install, and there run npm start
7. Now you should be set to go with the frontend and backend both running
